# 草案へのフィードバック

指定された草案ファイルを読み込み、フィードバック用のアイディアファイルを作成してください。

## 引数

$ARGUMENTS: フィードバック対象のファイルパス（例: `drafts/01.md`）

## 手順

1. 指定されたファイルを読み込む
2. `ideas/` に `YYYYMMDD-HHMMSS-feedback.md` の形式で新しいファイルを作成
3. 各行を引用形式に変換する（行番号は不要）
4. ユーザーにファイルを作成したことを伝える

## 出力フォーマット

各行を以下の形式で出力：

```
> {行の内容}
```

例：
```
> 吾輩は人間である。
> 名前もある。
```

## ファイルテンプレート

```markdown
---
created: YYYY-MM-DD HH:MM
status: new
type: feedback
source: {対象ファイルパス}
---

# フィードバック: {ファイル名}

以下の引用から、フィードバックしたい行だけを残してコメントを追記してください。
不要な行は削除してください。

---

> {行1}
> {行2}
...
```

## 注意

- 空行は引用に含めない（読みやすさのため）
- フロントマターの `type: feedback` と `source:` で通常のアイディアと区別する
- ユーザーが編集した後、`/reflect` で元ファイルと照合して反映される
