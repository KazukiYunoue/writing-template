# 進捗状況の確認

`drafts/` 配下の原稿ファイルの文字数・ページ数を集計し、進捗状況を表示します。

## 手順

1. `project.md` から目標文字数を取得（設定されている場合）
2. `drafts/` 内のすべてのマークダウンファイルを読み込む
3. 各ファイルの文字数を集計（メタデータ・空白を除く本文のみ）
4. ページ数を概算（400字/ページ または 800字/ページ）
5. 結果を表示

## 出力形式

```
## 進捗状況

### 全体
- 目標: XX,XXX字
- 現在: XX,XXX字（XX%）
- 概算ページ数: XX ページ（400字/ページ換算）

### ファイル別
| ファイル | 文字数 | 更新日 |
|----------|--------|--------|
| 01-序章.md | X,XXX字 | YYYY-MM-DD |
| 02-第一章.md | X,XXX字 | YYYY-MM-DD |
| ... | ... | ... |

### ステータス別
- draft: X ファイル
- review: X ファイル
- done: X ファイル
```

## 注意

- 文字数はマークダウン記法（#, *, - など）を除いた本文をカウント
- 目標文字数が `project.md` に未設定の場合は、達成率は表示しない
- ページ数は目安（400字詰め原稿用紙換算）
