# 原稿の草案作成

`outline.md` をもとに、`drafts/` ディレクトリに原稿ファイルを作成または執筆支援を行います。

## 対象の指定

$ARGUMENTS

- 引数がある場合：指定された章・セクションの草案を作成
- 引数がない場合：outline.md を確認し、次に執筆すべき箇所を提案

## 手順

1. `outline.md` を読み込み、構成を把握
2. 既存の `drafts/` 内のファイルを確認
3. 対象セクションの草案を作成、または執筆の続きを支援
4. 必要に応じて `research/` や `settings/` の情報を参照

## ファイル命名規則

```
drafts/
  01-章タイトル.md
  02-章タイトル.md
  ...
```

## 草案ファイルのテンプレート

```markdown
---
chapter: 1
title: 章タイトル
status: draft
updated: YYYY-MM-DD
---

# 章タイトル

（本文）
```

## 注意

- ユーザーの文体・トーンを尊重すること
- 草案はあくまで叩き台であり、ユーザーが自由に編集することを前提とする
- 勝手に物語の展開を決定しないこと。選択肢がある場合は提示する
- 既存の設定（settings/）との整合性を保つこと
