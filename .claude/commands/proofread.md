# 校正

原稿の誤字脱字・文法・表記の問題をチェックします。

## 対象の指定

$ARGUMENTS

- 引数がある場合：指定されたファイルを校正
- 引数がない場合：`drafts/` 内のすべてのファイルを対象とする

## チェック項目

### 誤字・脱字
- 漢字の誤変換
- タイプミス
- 脱字・衍字（余計な文字）

### 文法
- 「てにをは」の誤り
- 主語と述語のねじれ
- 修飾語の係り受け

### 表記の統一
- 漢字の開き閉じ（例：「出来る」vs「できる」）
- 送り仮名の統一
- 数字の表記（漢数字・アラビア数字）
- 敬体（です・ます）と常体（だ・である）の混在

### 句読点・記号
- 句読点の適切な使用
- 括弧の対応
- 全角・半角の統一

## 出力形式

```
## 校正結果：[ファイル名]

### 指摘事項

1. **行XX**: 「○○」→「△△」
   - 理由：誤変換

2. **行XX**: 「○○」→「△△」
   - 理由：主語と述語のねじれ

...

### 表記統一の確認

以下の表記が混在しています。どちらに統一しますか？
- 「出来る」（3箇所） / 「できる」（5箇所）

---

修正を適用しますか？（すべて / 選択 / キャンセル）
```

## 注意

- 必ず指摘理由を明示すること
- 修正は提案であり、ユーザーの承認を得ること
- 文体や表現の好みには踏み込まない（それは校閲の範囲）
