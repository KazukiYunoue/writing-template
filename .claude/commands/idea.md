# アイディアの発散

`ideas/` ディレクトリに新しいアイディアファイルを作成してください。

## 手順

1. 現在の日時を取得し、`ideas/YYYYMMDD-HHMMSS.md` の形式でファイルを作成
2. ファイルの冒頭に作成日時をメタデータとして記載
3. ユーザーにファイルを開いたことを伝え、アイディアの入力を促す

## ファイルテンプレート

```markdown
---
created: YYYY-MM-DD HH:MM
status: new
---

# アイディアメモ

（ここに自由に書き出してください）
```

## 注意

- ユーザーの入力をそのまま記録すること
- 整理や編集は `/reflect` コマンドで行うため、ここでは行わない
- 独り言や断片的なメモでも構わない
